# Sales-performance-Power-BI
The report that will help you understand the organization's current challenges and possible directions for improving the profitability of a large goods distributor's business.

# Financial overview page.

A visual element has been created that will show changes in sales levels, profits (in different segments) over a certain period of time.
The best products, sales distribution by region and product group, etc. have been demonstrated.

# Page for exploring metrics by month.

Specific figures for sales, expenses, profits at the level of each month are demonstrated.
Emphasis is placed on how an important parameter has changed compared to the previous period. 
Slicers of various attributes have been added for additional analysis capabilities.

# Discount Analysis Page.

Demonstrates how a sales discount affects the level of profit, namely how many orders there were for each discount level, how much they brought in in total volume and in net profit.

 # Sales by Region

 This page focuses on the geographical analysis of sales performance, regional profitability evaluation, and shipping cost monitoring.
